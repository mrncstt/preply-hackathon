# Demo Script (90 seconds)

## Before going on stage
- Laptop charged, presentation mode
- App pre-loaded at botas.vercel.app
- Mic permission pre-granted in browser
- Phone on silent

## Pitch flow (3 min total)
| Time | Slide | What to say |
|------|-------|-------------|
| 0:00 | Hook | "Right now Preply asks you to pick a tutor from a list..." |
| 0:10 | Problem | Let 73% land. "Not because learning is hard..." |
| 0:25 | Insight | "Research shows significantly better retention..." |
| 0:40 | Solution | "So we built an AI Discovery Coach..." |
| 0:55 | Why Now | "This was impossible 12 months ago..." |
| 1:05 | DEMO | Switch to app. Click Find your tutor. |
| 1:10 | Talk to AI | "I want to learn Spanish. I'm a beginner. I moved to Barcelona. I love football and La Liga." |
| 1:50 | Show output | "Look at the Learning Bridge..." |
| 2:05 | Thymia | "In production, Thymia adds cognitive signals..." |
| 2:15 | Business | "Better matching, higher retention, lower churn." |
| 2:25 | Tech | "Next.js, Agora, OpenAI, Thymia, Vercel. Built in 24h." |
| 2:35 | Close | "Stop teaching languages. Start discovering learners." |
| 2:47 | Done | Silence. Let it land. |

## If demo fails
Say: "Live demos and conference WiFi -- you know how it goes."
Show pre-recorded video (save locally!).

## What to say to the AI agent
Keep answers SHORT:
- "I want to learn Spanish"
- "Complete beginner, I just moved to Barcelona"
- "I need it for work meetings"
- "I'm obsessed with football, I watch La Liga every weekend"
- "My biggest fear is sounding stupid in meetings"

## Top 5 judge questions
1. "How is this different from ChatGPT?" -> "Voice-first, structured output, cognitive signals"
2. "Would Preply implement this?" -> "Slots into existing funnel, enhances onboarding"
3. "Thymia is mocked?" -> "Pipeline built, plug-and-play when API available"
4. "Cost?" -> "$0.08-0.12 per session"
5. "Next steps?" -> "Wire Thymia real, tutor dashboard, A/B test"
