2. Screen-by-Screen Interface Comparison
   (LogRocket)

Compara tela a tela os fluxos criticos
entre concorrentes:

- Onboarding: quantos passos, friccao,
  tempo ate primeiro valor
- Core loop: como o usuario executa a
  tarefa principal
- Feature discovery: como features de IA
  sao apresentadas
- Feedback loops: como o app comunica
  progresso

Entregavel: screenshots lado a lado com
anotacoes de UX

5. SWOT + Feature Checklist + User Flow
   (UXtweak)

Framework em 3 camadas:

1. Feature Checklist: lista de features
   com checkmark por concorrente
2. User Flow Mapping: mapeamento dos
   fluxos criticos (signup -> primeira aula
   -> feedback -> retencao)
3. SWOT por concorrente: forcas,
   fraquezas, oportunidades, ameacas

Template Google Sheets disponivel:
https://blog.uxtweak.com/competitive-analy
sis-in-ux-research/
