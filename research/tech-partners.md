# Tech Partners - Hackathon

## Agora - Real-Time Audio/Video + Conversational AI

- **Core**: SDK de audio/video real-time, 80B min/mes em 200+ paises
- **Conversational AI Engine** (marco 2025):
  - Integra com qualquer LLM + qualquer TTS/ASR
  - Ultra-low latency para voice AI
  - Natural interruption handling (modelo pausa quando usuario interrompe)
  - AI echo cancellation + noise suppression
  - Ganhou Microsoft AI Innovation Award no CES 2025
- **Agora + OpenAI SDK**: Integracao direta com OpenAI Realtime API
  - Speech processado diretamente (nao convertido para texto primeiro)
  - Entende emocao humana pela voz
  - Language education listado como use case alvo
- **Para o hackathon**: Ponte entre modelos AI e interacao humana real-time

## OpenAI - Capacidades Mais Recentes

- **gpt-realtime** (GA ago 2025):
  - Streaming audio in/out
  - Interruption handling com voice activity detection
  - Background function calling enquanto modelo continua falando
  - Instruction following: 82.8% (antes 65.6%)
  - Pricing: $32/1M audio input tokens, $64/1M output tokens

- **GPT-5.2 / Codex**: Reasoning para code gen e tasks multi-step
- **Transcription API**: Funciona com fala fragmentada, com sotaque, nao-nativa
- **Para o hackathon**: O "cerebro" AI - conversacao, reasoning, speech, geracao de conteudo

## AWS - Servicos AI/ML Relevantes

- **Amazon Bedrock**: Acesso gerenciado a foundation models com RAG, agents, guardrails
- **Amazon Transcribe**: Speech-to-text real-time, multi-idioma
- **Amazon Translate**: Traducao real-time
- **Amazon Polly**: TTS com vozes generativas
- **Pipeline combinado**: Transcribe -> Translate -> Polly = speech-to-speech near-real-time
- **Pipecat integration**: Framework para voice AI agents no Bedrock
- **Para o hackathon**: Camada de infra, scaling e servicos supplementares

## Anam - Avatares AI

- **Core**: API de avatar AI interativo em tempo real
- **CARA-3**: Geracao fotorrealista de avatares

  - Lip-sync perfeito

  - **180ms de latencia**

- **Para o hackathon**: Embodiment visual para tutores AI. Transforma voz em conversa face-to-face

## Thymia - Biomarcadores de Voz e Comportamento

- **Core**: Avaliacao de saude mental via biomarcadores de voz, comportamento facial, jogos cognitivos
- Detecta sinais de saude a partir de **15 segundos de fala**
- Avalia **30+ estados clinicos e de bem-estar** com 85%+ de acuracia
- Analisa: padroes de fala, micro-expressoes, dinamica do olhar, biomarcadores vocais
- Dataset proprietario: 1B+ data points de 5000+ individuos
- **Helios API**: Analise de biomarcadores vocais para wellness
- Primeira tecnologia de biomarcador vocal regulamentada do mundo
- **Para o hackathon**: Medir estado emocional, stress, confianca, carga cognitiva do learner pela voz. Detectar frustracao, ansiedade, desengajamento para respostas adaptativas
