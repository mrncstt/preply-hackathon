# Hackathon Thought Starters (Official)

Source: Preply x Agora Hackathon guide

## 1. Language Anxiety Wellness Coach
- Anam avatar + Thymia biomarkers
- Detects stress, adjusts pace, switches to easier material
- Avatar provides calming visual presence
- **Tech:** Agora + Anam + Thymia

## 2. Pronunciation Practice with Voice Biomarkers
- Thymia voice analysis beyond speech recognition
- Track vocal confidence, detect frustration
- Targeted exercises + celebrates improvement
- **Tech:** Agora + Thymia

## 3. AI Avatar Language Tutor
- Face-to-face with Anam video avatars
- Facial expressions for non-verbal feedback
- Adapts difficulty based on responses
- Multiple languages and cultural contexts
- **Tech:** Agora + Anam + OpenAI

## 4. Mock Interview Prep in Any Language
- Professional conversations in target language
- Thymia tracks stress and confidence
- Feedback on vocal clarity, hesitation, composure
- **Tech:** Agora + Thymia + OpenAI

## 5. Immersive Cultural Conversation Partner
- Real-world scenarios (restaurant, directions, market)
- Different characters, adjustable difficulty
- Cultural context alongside language corrections
- **Tech:** Agora + OpenAI

## 6. Learning Progress Dashboard with AI Analytics
- Tracks sessions over time
- Conversation transcripts + Thymia biomarker data
- Vocabulary growth, fluency, confidence trends
- Personalizes each session from data
- **Tech:** Agora + Thymia + OpenAI

## 7. Group Language Exchange Facilitator
- Multi-party voice/video via Agora RTC
- AI moderates, suggests topics, corrects errors
- RTM for text-based vocabulary hints
- Balanced participation
- **Tech:** Agora RTC + RTM + OpenAI
